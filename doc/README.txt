The hl parser tool is used to create game books for games like Sherlock Holmes Consulting Detective.
It was created to support a new game system calleed "High & Low".

This is a very early alpha version.

For more information contact jessereichler@gmail.com

For now, the best documentation can be found in the test document.

INSTRUCTIONS FOR USE:
Create a directory for each case, modeled after the sample case.
Leads can be spread over multiple .txt files.

REQUIREMENTS:
Mistletoe library (for markdown)


SYNTAX:
The markdown syntax is used for leads.
Start each lead with # LEADID or # LEADID: LABEL
See examples for more complicated use of code within leads
